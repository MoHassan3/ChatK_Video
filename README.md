App Features Outline

1. User Authentication and Profile
Sign up and log in using email/password or Google. Update profile with a photo.

2. Real-time Data and Messaging 
Real-time chat with Cloud Firestore. Add emojis to messages.

3. Media Management
Upload and display images with caching. Save and share media.

4. File and Document Handling 
Pick and upload files. View PDFs in the app.

5. Permissions and Contacts
Manage permissions for contacts and storage. Import and manage contacts.
 
6.Storage Found in all phones with email only 

7. Network and Backend Interaction 
Fetch and post data to backend services.

8. User Experience Enhancements 
Show toast messages for feedback. Customize app launcher icons.

9. Show Only Known Users or Contacts
Access and match device contacts with registered users.

10. Google Sign-In (No Account or Mobile Number Needed)
Sign in with Google account.

11. Adding Users Using Email ID


12. Simple User Profile (with Photo, About, Last-seen, Created-at)
Store user profile details and photo.

13. Message Read Status
Track and show read receipts.

14. Message Delete & Edit Feature
Edit or delete messages.

15. Decent UI with New Material Widgets
Use Material Design for a modern UI.
